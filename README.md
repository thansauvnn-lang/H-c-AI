# Web demo phát hiện giao dịch bất thường

Đây là một trang web demo đơn giản, mô phỏng cách phát hiện giao dịch rút tiền bất thường
dựa trên các quy tắc cơ bản (không dùng AI/ML).

## Mục đích
- Học làm web (HTML/CSS/JS)
- Minh họa tư duy phát hiện bất thường
- Làm nền để phát triển sau này (nếu muốn)

## Cách sử dụng
- Mở file `index.html` bằng trình duyệt
- Xem danh sách giao dịch giả lập
- Hệ thống đánh dấu giao dịch bình thường / nghi ngờ

## Ghi chú
- Đây là bản demo học tập
- Logic hiện tại là rule-based, không phải AI thật
